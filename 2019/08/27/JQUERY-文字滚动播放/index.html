2019-08-27
  • 前端相关
  • Jquery

文字轮流滚动播放


思路：

HTML & CSS

1.先建一个容器,规定高度，宽度100%,关键是overflow:hidden
2.里面的li必须垂直居中，不然有可能会显示下一个li的一点点+浮动就可以了
3.图像保持垂直居中有很多方式,比如通过在img标签设置align属性,也可以通过css的vertical-align:middle也可以,总之方法很多…

JQUERTY

1.设置容器里面ul的过渡动画,通过animate来设置margin的变化从而实现向上滚动的效果,在通过callback来使ul的margin返回原本的margin数值,把第一个li放到ul最后的位置，来达到无限重复往上滚动
2.设置定时器，先判断是否有li,如果有,则设置定时器来调用上面的函数
