[JQUERY]简单加载画面

2019-08-28
  • 前端相关
  • Jquery

简单实现页面加载时候的读取画面,可以用在加载资源过久时使用…


思路：

HTML & CSS

简单点理解就是建立一个div来实现遮罩层，其中重点就在于遮罩层容器设置绝对定位,不然就要设置你想要显示的内容隐藏起来等触发在让他显示，在这里选择通过css简单实现，并且遮罩层的层级要比内容高

JQUERTY

1.这个就更简单了，由于一开始遮罩层在css里面设置了隐身，所以需要文档加载时，就要设置display:block让他显示
2.等全部加载完成在运行一个函数，通过delay+fadeout来使遮罩层容器以及内容分别隐藏，最后在设置让主要内容显示
3.防止经过一段时间仍未加载的情况，留一手强制显示内容，也就是通过定时器触发上面同样功能的函数即可。
